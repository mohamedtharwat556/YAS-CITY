# YAS Company Website

موقع ويب احترافي لشركة YAS تم تصميمه باستخدام HTML و CSS و JavaScript.

## المميزات

- 🎨 تصميم عصري وجذاب
- 📱 متجاوب مع جميع الأجهزة
- 🌐 دعم كامل للغة العربية
- ✨ تأثيرات حركية سلسة
- 🖼️ معرض صور تفاعلي
- 📝 نموذج اتصال فعال
- 🧭 قائمة تنقل ثابتة

## الهيكلية

```
yas-website/
├── index.html      # الصفحة الرئيسية
├── style.css       # تنسيقات CSS
├── script.js       # وظائف JavaScript
├── team-photo.jpg  # صورة فريق العمل
├── building-photo.jpg # صورة المقر الرئيسي
└── README.md       # ملف القراءة
```

## الأقسام

1. **الرئيسية** - قسم الترحيب مع عرض الصور
2. **من نحن** - معلومات عن الشركة
3. **خدماتنا** - عرض الخدمات المقدمة
4. **معرض الصور** - معرض تفاعلي للصور
5. **اتصل بنا** - نموذج الاتصال ومعلومات التواصل

## التقنيات المستخدمة

- HTML5
- CSS3 مع Flexbox و Grid
- JavaScript (ES6+)
- Google Fonts (Tajawal)
- تصميم متجاوب (Responsive Design)

## التشغيل

1. افتح المجلد في متصفح الويب
2. أو استخدم خادم محلي لتشغيل الموقع

```bash
# باستخدام Python
python -m http.server 8000

# باستخدام Node.js
npx serve .
```

## التخصيص

يمكن تخصيص الموقع بسهولة:
- تعديل الألوان في ملف `style.css`
- تغيير المحتوى في ملف `index.html`
- إضافة وظائف جديدة في ملف `script.js`
